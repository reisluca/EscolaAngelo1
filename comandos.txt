# Como vincular um repositório do GitHub com o Visual Studio Code
1 No GitHub, abra o repositório e clique no botão CODE -> Local -> copiar link do repositório
2 No Visual Studio Code, abra a pasta de seus projetos ( Documentos/Python/ )
3 Abra o Terminal e digite o comando: git clone <cole aqui o link copiado>
4 No Terminal, digite o comando: git config user.name <digite aqui seu nome de usuário no GitHub>
5 Ainda no Terminal, digite o comando: git config user.email <digite aqui seu e-mail no GitHub>
